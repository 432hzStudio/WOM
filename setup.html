Instrucciones para configurar WOM Argentina

Nota: Este proyecto requiere Node.js y npm instalados. Asegúrate de tenerlos antes de continuar.

Paso 1: Configurar el Frontend

Navega a la carpeta del frontend e instala las dependencias:

cd frontend
npm install

Paso 2: Configurar el Backend

Navega a la carpeta del backend e instala las dependencias:

cd backend
npm install

Paso 3: Configurar la Base de Datos

Configura PostgreSQL y asegúrate de que las credenciales coincidan con las del archivo backend/src/config/config.js

Paso 4: Iniciar el Backend

En una terminal, navega a la carpeta del backend e inicia el servidor:

cd backend
npm start

Paso 5: Iniciar el Frontend

En otra terminal, navega a la carpeta del frontend e inicia la aplicación React:

cd frontend
npm start

Esto abrirá la aplicación en tu navegador automáticamente, generalmente en http://localhost:3000

Método alternativo: Abrir directamente

Si prefieres ver la aplicación sin configuración completa, puedes abrir el siguiente archivo en tu navegador:

frontend/public/index.html

Pero ten en cuenta que algunas funcionalidades no estarán disponibles sin el backend en funcionamiento.
